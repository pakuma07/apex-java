## Problem 111: Unique Paths
**LeetCode #62** | Count paths from top-left to bottom-right (right/down moves only).

### Brute Force -- O(2^(M+N)) recursion
```java
int uniquePathsR(int m, int n) { if (m == 1 || n == 1) return 1; return uniquePathsR(m - 1, n) + uniquePathsR(m, n - 1); }
```

### Optimal -- O(M*N) DP (or O(min(M,N)) with math: C(m+n-2, m-1))
```java
int uniquePaths(int m, int n) {
    int[][] dp = new int[m][n];
    for (int[] row : dp) Arrays.fill(row, 1);
    for (int r = 1; r < m; r++)
        for (int c = 1; c < n; c++) dp[r][c] = dp[r - 1][c] + dp[r][c - 1];
    return dp[m - 1][n - 1];
}
```

## Problem 112: Longest Common Subsequence
**LeetCode #1143**

### Brute Force -- O(2^(M+N)) recursion
### Optimal -- O(M*N) DP
```java
int longestCommonSubsequence(String a, String b) {
    int m = a.length(), n = b.length();
    int[][] dp = new int[m + 1][n + 1];
    for (int i = 1; i <= m; i++)
        for (int j = 1; j <= n; j++)
            dp[i][j] = (a.charAt(i - 1) == b.charAt(j - 1))
                ? dp[i - 1][j - 1] + 1
                : Math.max(dp[i - 1][j], dp[i][j - 1]);
    return dp[m][n];
}
```

## Problem 113: Best Time to Buy and Sell Stock with Cooldown
**LeetCode #309** | Cooldown of 1 day after selling.

### Brute Force -- O(2^N) try all buy/sell sequences
### Optimal -- O(N) state machine DP
```java
int maxProfit(int[] prices) {
    int hold = Integer.MIN_VALUE, sold = 0, rest = 0;
    for (int p : prices) {
        int ph = hold, ps = sold, pr = rest;
        hold = Math.max(ph, pr - p);   // buy today or stay holding
        sold = ph + p;                 // sell today
        rest = Math.max(pr, ps);       // rest or coming off cooldown
    }
    return Math.max(sold, rest);
}
```

## Problem 114: Coin Change II
**LeetCode #518** | Count combinations to make amount (unlimited use).

### Brute Force -- O(amount^coins) recursion
### Optimal -- O(coins * amount) unbounded knapsack DP
```java
int change(int amount, int[] coins) {
    int[] dp = new int[amount + 1];
    dp[0] = 1;
    for (int c : coins)
        for (int i = c; i <= amount; i++) dp[i] += dp[i - c];
    return dp[amount];
}
```

## Problem 115: Target Sum
**LeetCode #494** | Assign + or - to each number; count ways to reach target.

### Brute Force -- O(2^N) try all + and - assignments
```java
int findTargetSumWaysR(int[] n, int i, int t) {
    if (i == n.length) return t == 0 ? 1 : 0;
    return findTargetSumWaysR(n, i + 1, t - n[i]) + findTargetSumWaysR(n, i + 1, t + n[i]);
}
int findTargetSumWays(int[] nums, int target) { return findTargetSumWaysR(nums, 0, target); }
```

### Optimal -- O(N * sum) DP with offset
```java
int findTargetSumWays(int[] nums, int target) {
    int sum = 0;
    for (int x : nums) sum += x;
    if (Math.abs(target) > sum) return 0;
    Map<Integer, Integer> dp = new HashMap<>();
    dp.put(0, 1);
    for (int x : nums) {
        Map<Integer, Integer> next = new HashMap<>();
        for (Map.Entry<Integer, Integer> e : dp.entrySet()) {
            int s = e.getKey(), c = e.getValue();
            next.merge(s + x, c, Integer::sum);
            next.merge(s - x, c, Integer::sum);
        }
        dp = next;
    }
    return dp.getOrDefault(target, 0);
}
```

## Problem 116: Interleaving String
**LeetCode #97** | Can s3 be formed by interleaving s1 and s2?

### Brute Force -- O(2^(M+N)) recursion
### Optimal -- O(M*N) DP
```java
boolean isInterleave(String s1, String s2, String s3) {
    int m = s1.length(), n = s2.length();
    if (m + n != s3.length()) return false;
    boolean[][] dp = new boolean[m + 1][n + 1];
    dp[0][0] = true;
    for (int i = 1; i <= m; i++) dp[i][0] = dp[i - 1][0] && s1.charAt(i - 1) == s3.charAt(i - 1);
    for (int j = 1; j <= n; j++) dp[0][j] = dp[0][j - 1] && s2.charAt(j - 1) == s3.charAt(j - 1);
    for (int i = 1; i <= m; i++)
        for (int j = 1; j <= n; j++)
            dp[i][j] = (dp[i - 1][j] && s1.charAt(i - 1) == s3.charAt(i + j - 1))
                    || (dp[i][j - 1] && s2.charAt(j - 1) == s3.charAt(i + j - 1));
    return dp[m][n];
}
```

## Problem 117: Longest Increasing Path in Matrix
**LeetCode #329**

### Brute Force -- O(M*N * 4^(M*N)) DFS from every cell without memoization
### Optimal -- O(M*N) DFS with memoization
```java
int[] dx = {1, -1, 0, 0}, dy = {0, 0, 1, -1};

int longestIncreasingPath(int[][] mat) {
    int m = mat.length, n = mat[0].length, res = 0;
    int[][] memo = new int[m][n];
    for (int r = 0; r < m; r++)
        for (int c = 0; c < n; c++) res = Math.max(res, dfs(mat, r, c, memo));
    return res;
}
int dfs(int[][] mat, int r, int c, int[][] memo) {
    if (memo[r][c] != 0) return memo[r][c];
    int m = mat.length, n = mat[0].length, best = 1;
    for (int d = 0; d < 4; d++) {
        int nr = r + dx[d], nc = c + dy[d];
        if (nr >= 0 && nr < m && nc >= 0 && nc < n && mat[nr][nc] > mat[r][c])
            best = Math.max(best, 1 + dfs(mat, nr, nc, memo));
    }
    return memo[r][c] = best;
}
```

## Problem 118: Distinct Subsequences
**LeetCode #115** | Count distinct ways s contains t as subsequence.

### Brute Force -- O(2^M) try all subsequences of s
### Optimal -- O(M*N) DP
```java
int numDistinct(String s, String t) {
    int m = s.length(), n = t.length();
    long[][] dp = new long[m + 1][n + 1];
    for (int i = 0; i <= m; i++) dp[i][0] = 1;
    for (int i = 1; i <= m; i++)
        for (int j = 1; j <= n; j++) {
            dp[i][j] = dp[i - 1][j];
            if (s.charAt(i - 1) == t.charAt(j - 1)) dp[i][j] += dp[i - 1][j - 1];
        }
    return (int) dp[m][n];
}
```

## Problem 119: Edit Distance
**LeetCode #72** | Min operations (insert, delete, replace) to transform word1 to word2.

### Brute Force -- O(3^(M+N)) recursion
### Optimal -- O(M*N) DP
```java
int minDistance(String w1, String w2) {
    int m = w1.length(), n = w2.length();
    int[][] dp = new int[m + 1][n + 1];
    for (int i = 0; i <= m; i++) dp[i][0] = i;
    for (int j = 0; j <= n; j++) dp[0][j] = j;
    for (int i = 1; i <= m; i++)
        for (int j = 1; j <= n; j++) {
            if (w1.charAt(i - 1) == w2.charAt(j - 1)) dp[i][j] = dp[i - 1][j - 1];
            else dp[i][j] = 1 + Math.min(dp[i - 1][j - 1], Math.min(dp[i - 1][j], dp[i][j - 1]));
        }
    return dp[m][n];
}
```

## Problem 120: Burst Balloons
**LeetCode #312** | Interval DP: maximize coins from bursting balloons.

### Brute Force -- O(N! * N) try all permutations of bursting order
### Optimal -- O(N^3) interval DP (think: which balloon is burst LAST in interval)
```java
int maxCoins(int[] nums) {
    int n = nums.length + 2;
    int[] arr = new int[n];
    arr[0] = arr[n - 1] = 1;
    for (int i = 0; i < nums.length; i++) arr[i + 1] = nums[i];
    int[][] dp = new int[n][n];
    for (int len = 2; len < n; len++) {
        for (int l = 0; l < n - len; l++) {
            int r = l + len;
            for (int k = l + 1; k < r; k++)
                dp[l][r] = Math.max(dp[l][r], dp[l][k] + arr[l] * arr[k] * arr[r] + dp[k][r]);
        }
    }
    return dp[0][n - 1];
}
```

## Problem 121: Regular Expression Matching
**LeetCode #10** | Match pattern with '.' (any char) and '*' (0+ of preceding).

### Brute Force -- O(2^(M+N)) recursion without memoization
### Optimal -- O(M*N) DP
```java
boolean isMatch(String s, String p) {
    int m = s.length(), n = p.length();
    boolean[][] dp = new boolean[m + 1][n + 1];
    dp[0][0] = true;
    for (int j = 2; j <= n; j++) if (p.charAt(j - 1) == '*') dp[0][j] = dp[0][j - 2];
    for (int i = 1; i <= m; i++)
        for (int j = 1; j <= n; j++) {
            if (p.charAt(j - 1) == '*') {
                dp[i][j] = dp[i][j - 2]; // use '*' as 0 occurrences
                if (p.charAt(j - 2) == '.' || p.charAt(j - 2) == s.charAt(i - 1))
                    dp[i][j] = dp[i][j] || dp[i - 1][j];
            } else {
                dp[i][j] = (p.charAt(j - 1) == '.' || p.charAt(j - 1) == s.charAt(i - 1)) && dp[i - 1][j - 1];
            }
        }
    return dp[m][n];
}
```
