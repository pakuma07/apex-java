## Problem 99: Climbing Stairs
**LeetCode #70** | N stairs, 1 or 2 steps at a time. Count ways.

### Brute Force -- O(2^N) recursion without memoization
```java
int climbStairs(int n) { if (n <= 2) return n; return climbStairs(n - 1) + climbStairs(n - 2); }
```

### Optimal -- O(N) time, O(1) space Fibonacci
```java
int climbStairs(int n) {
    int a = 1, b = 2;
    for (int i = 3; i <= n; i++) { int c = a + b; a = b; b = c; }
    return n == 1 ? 1 : b;
}
```

## Problem 100: Min Cost Climbing Stairs
**LeetCode #746** | Pay cost[i] to step from stair i; start from 0 or 1.

### Brute Force -- O(2^N) recursion
```java
int minCostClimbingStairsMemo(int[] c, int i, int[] dp) {
    if (i >= c.length) return 0;
    if (dp[i] != -1) return dp[i];
    return dp[i] = c[i] + Math.min(minCostClimbingStairsMemo(c, i + 1, dp),
                                   minCostClimbingStairsMemo(c, i + 2, dp));
}
int minCostClimbingStairs(int[] cost) {
    int n = cost.length;
    int[] dp = new int[n];
    Arrays.fill(dp, -1);
    return Math.min(minCostClimbingStairsMemo(cost, 0, dp), minCostClimbingStairsMemo(cost, 1, dp));
}
```

### Optimal -- O(N) time, O(1) space
```java
int minCostClimbingStairs(int[] cost) {
    int n = cost.length;
    for (int i = 2; i < n; i++) cost[i] += Math.min(cost[i - 1], cost[i - 2]);
    return Math.min(cost[n - 1], cost[n - 2]);
}
```

## Problem 101: House Robber
**LeetCode #198** | Max sum of non-adjacent elements.

### Brute Force -- O(2^N) try all subsets
### Optimal -- O(N) time, O(1) space
```java
int rob(int[] nums) {
    int a = 0, b = 0;
    for (int x : nums) { int c = Math.max(b, a + x); a = b; b = c; }
    return b;
}
```

int robLinear(int[] v, int l, int r) {

## Problem 103: Longest Palindromic Substring
**LeetCode #5**

### Brute Force -- O(N^3) check all substrings
```java
boolean isPalin(String s, int l, int r) {
    while (l < r) if (s.charAt(l++) != s.charAt(r--)) return false;
    return true;
}
String longestPalindromeBf(String s) {
    String res = "";
    for (int i = 0; i < s.length(); i++)
        for (int j = i; j < s.length(); j++)
            if (isPalin(s, i, j) && j - i + 1 > res.length()) res = s.substring(i, j + 1);
    return res;
}
```

### Optimal -- O(N^2) expand around center
```java
int start = 0, best = 1;

String longestPalindrome(String s) {
    int n = s.length();
    start = 0; best = 1;
    for (int i = 0; i < n; i++) { expand(s, i, i); expand(s, i, i + 1); }
    return s.substring(start, start + best);
}
void expand(String s, int l, int r) {
    int n = s.length();
    while (l >= 0 && r < n && s.charAt(l) == s.charAt(r)) {
        if (r - l + 1 > best) { best = r - l + 1; start = l; }
        l--; r++;
    }
}
```

## Problem 104: Palindromic Substrings
**LeetCode #647** | Count all palindromic substrings.

### Brute Force -- O(N^3) check all substrings
### Optimal -- O(N^2) expand around center
```java
int cnt = 0;

int countSubstrings(String s) {
    int n = s.length();
    cnt = 0;
    for (int i = 0; i < n; i++) { expand(s, i, i); expand(s, i, i + 1); }
    return cnt;
}
void expand(String s, int l, int r) {
    int n = s.length();
    while (l >= 0 && r < n && s.charAt(l) == s.charAt(r)) { cnt++; l--; r++; }
}
```

## Problem 105: Decode Ways
**LeetCode #91** | Count decodings of digit string (1=A...26=Z).

### Brute Force -- O(2^N) recursion
### Optimal -- O(N) DP
```java
int numDecodings(String s) {
    int n = s.length();
    int[] dp = new int[n + 1];
    dp[n] = 1;
    for (int i = n - 1; i >= 0; i--) {
        if (s.charAt(i) == '0') { dp[i] = 0; continue; }
        dp[i] = dp[i + 1];
        if (i + 1 < n && (s.charAt(i) - '0') * 10 + (s.charAt(i + 1) - '0') <= 26) dp[i] += dp[i + 2];
    }
    return dp[0];
}
```

## Problem 106: Coin Change
**LeetCode #322** | Min coins to make amount (unlimited use).

### Brute Force -- O(S^C) recursion without memoization
### Optimal -- O(S * C) bottom-up DP
```java
int coinChange(int[] coins, int amount) {
    int[] dp = new int[amount + 1];
    Arrays.fill(dp, amount + 1);
    dp[0] = 0;
    for (int i = 1; i <= amount; i++)
        for (int c : coins)
            if (c <= i) dp[i] = Math.min(dp[i], 1 + dp[i - c]);
    return dp[amount] > amount ? -1 : dp[amount];
}
```

## Problem 107: Maximum Product Subarray
**LeetCode #152**

### Brute Force -- O(N^2) try all subarrays
### Optimal -- O(N) track both min and max product
```java
int maxProduct(int[] nums) {
    int mx = nums[0], mn = nums[0], res = nums[0];
    for (int i = 1; i < nums.length; i++) {
        if (nums[i] < 0) { int t = mx; mx = mn; mn = t; }
        mx = Math.max(nums[i], mx * nums[i]);
        mn = Math.min(nums[i], mn * nums[i]);
        res = Math.max(res, mx);
    }
    return res;
}
```

## Problem 108: Word Break
**LeetCode #139** | Can s be segmented into dictionary words?

### Brute Force -- O(2^N) recursion checking all splits
### Optimal -- O(N^2) DP with hash set
```java
boolean wordBreak(String s, List<String> wordDict) {
    Set<String> ws = new HashSet<>(wordDict);
    int n = s.length();
    boolean[] dp = new boolean[n + 1];
    dp[0] = true;
    for (int i = 1; i <= n; i++)
        for (int j = 0; j < i; j++)
            if (dp[j] && ws.contains(s.substring(j, i))) { dp[i] = true; break; }
    return dp[n];
}
```

## Problem 109: Longest Increasing Subsequence
**LeetCode #300**

### Brute Force -- O(N^2) DP
```java
int lengthOfLISDp(int[] nums) {
    int n = nums.length, res = 1;
    int[] dp = new int[n];
    Arrays.fill(dp, 1);
    for (int i = 1; i < n; i++) {
        for (int j = 0; j < i; j++) if (nums[j] < nums[i]) dp[i] = Math.max(dp[i], dp[j] + 1);
        res = Math.max(res, dp[i]);
    }
    return res;
}
```

### Optimal -- O(N log N) patience sort (binary search on tails array)
```java
int lengthOfLIS(int[] nums) {
    List<Integer> tails = new ArrayList<>();
    for (int x : nums) {
        int idx = lowerBound(tails, x);
        if (idx == tails.size()) tails.add(x);
        else tails.set(idx, x);
    }
    return tails.size();
}
int lowerBound(List<Integer> tails, int x) {
    int lo = 0, hi = tails.size();
    while (lo < hi) {
        int mid = (lo + hi) / 2;
        if (tails.get(mid) < x) lo = mid + 1;
        else hi = mid;
    }
    return lo;
}
```

## Problem 110: Partition Equal Subset Sum
**LeetCode #416** | Can array be partitioned into two equal-sum subsets?

### Brute Force -- O(2^N) try all subsets
### Optimal -- O(N * S) 0/1 knapsack DP with bitset trick
```java
boolean canPartition(int[] nums) {
    int sum = 0;
    for (int x : nums) sum += x;
    if (sum % 2 != 0) return false;
    int target = sum / 2;
    boolean[] dp = new boolean[target + 1];
    dp[0] = true;
    for (int x : nums)
        for (int j = target; j >= x; j--) dp[j] = dp[j] || dp[j - x];
    return dp[target];
}
```
